# Elektrophon (1921) — Cmajor Emulation

A historically grounded emulation of Jörg Mager’s Elektrophon (1921), one of the earliest heterodyne electronic instruments.

## Features

- True heterodyne-style synthesis (RF oscillator interaction)
- Continuous pitch (no keyboard)
- Microtonal resolution
- Crank-style inertia modeling
- Analog drift and instability
- Primitive timbre shaping

## Controls

| Parameter      | Description |
|----------------|------------|
| pitchControl   | Continuous pitch (0–1) |
| ampControl     | Amplitude (expression pedal style) |
| timbre         | Subtle tone shaping |

## Philosophy

This is NOT a traditional synthesizer.

It is designed as:
- A continuous performance instrument
- A microtonal sound generator
- An expressive system closer to voice or violin

## Usage

Best used with:
- Ribbon controller
- MPE surface
- High-resolution knob + expression pedal
